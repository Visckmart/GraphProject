O algoritmo DFS usa uma pilha como estrutura de controle
pilha = Pilha()
Inserindo o nó inicial na pilha para ser o primeiro analisado
pilha.push(nóInicial)

enquanto tamanho da pilha > 0 {
    Escolhe o nó no topo da pilha como nó corrente
    e o marca como visitado.
    nóCorrente = pilha.pop()
    nóCorrente.visitado = true

    para cada nó vizinho de nóCorrente {
        se nó não visitado {
            Insere o nó corrente na pilha para
            retornar à ele caso haja mais vizinhos.
            pilha.push(nóCorrente)

            Insere nó na pilha para visitá-lo em seguida.
            pilha.push(nó)

            volta ao início do primeiro loop
        }
    }

}
