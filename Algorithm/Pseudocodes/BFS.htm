O algoritmo DFS usa uma fila como estrutura de controle
fila = Fila()
Inserindo o nó inicial na fila para ser o primeiro analisado
fila.push(nóInicial)

enquanto tamanho da fila > 0 {
    Escolhe o nó no fim da fila como nó corrente
    e o marca como visitado.
    nóCorrente = fila.remove()
    nóCorrente.visitado = true

    para cada nó vizinho de nóCorrente {
        se nó não visitado {
            Insere o nó na fila para visitá-lo posteriormente
            fila.insere(nó)

            Marca nó como visitado ao ser descoberto
            nó.visitado = true
        }
    }

}
